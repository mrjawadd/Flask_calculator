**Description:**  
A **web-based calculator** with a modern, responsive UI that supports basic and advanced mathematical operations.  
**Key Features:**  
- Addition, subtraction, multiplication, division, and more  
- Interactive, easy-to-use interface  
- Mobile-friendly and responsive design  
**Tech Stack:** Python, Flask, HTML/CSS, JavaScript  
**Potential Improvements:** Add scientific calculator features like trigonometry and logarithms.

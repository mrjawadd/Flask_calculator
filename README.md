# Interactive Calculator

A **web-based calculator** with a modern and responsive interface that supports both basic and advanced mathematical operations. Designed for quick calculations and user-friendly interaction, it works seamlessly on desktops and mobile devices.

## Overview

The Interactive Calculator allows users to perform arithmetic operations such as addition, subtraction, multiplication, and division, as well as more advanced calculations if extended. Its responsive design ensures smooth functionality across different screen sizes.

## Key Features

* **Basic Operations:** Addition, subtraction, multiplication, and division.
* **Advanced Operations:** Easily extendable to include scientific calculations like powers, roots, and logarithms.
* **Responsive Design:** Mobile-friendly interface for use on any device.
* **Interactive UI:** Clickable buttons for seamless input and instant results.

## How It Works

1. Open the calculator in a web browser.
2. Use the on-screen buttons or keyboard to input numbers and operators.
3. Click **equals** to compute the result instantly.
4. Clear or reset as needed for new calculations.

## Tech Stack

* **Backend:** Python with Flask framework
* **Frontend:** HTML, CSS, JavaScript for interactivity and design
* **Responsive Layout:** Works across desktops, tablets, and mobile devices

## Installation and Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/mrjawadd/Flask_calculator.git
   cd Flask_calculator
   ```
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Run the system:

   ```bash
   python app.py
   ```
4. Open your browser and navigate to:

   ```
   http://127.0.0.1:5000
   ```
5. Start performing calculations immediately.
